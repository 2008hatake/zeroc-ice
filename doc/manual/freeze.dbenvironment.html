Freeze::DBEnvironment

Overview

local interface DBEnvironment

A database environment. Multiple databases can be created from a single database environment. The database environment also offers operations to create transactions.

Used By

::IceBox::FreezeService::start

See Also

DBTransaction, DB

Operation Index

close

Close the database environment and destroy this database environment object.

getCommunicator

Get the communicator for this database environment.

getName

Get the name of the database environment.

openDB

Open and return a database object.

openDBWithTxn

Open and return a database object within the context of a transaction.

startTransaction

Start a new transaction in this database environment, and return the transaction object.

sync

Flush any cached information to the disk.

close

void close()
    throws
	DBException;

Close the database environment and destroy this database environment object. This operation also calls close on all databases that have been opened with this database environment object. Subsequent calls to close have no effect.

Exceptions

DBException

Raised if a database failure occurred.

See Also

DB::close

getCommunicator

::Ice::Communicator getCommunicator();

Get the communicator for this database environment.

Return Value

The communicator for this database environment.

getName

string getName();

Get the name of the database environment.

Return Value

The name of this database environment.

openDB

DB openDB(bool create,
    string name)
    throws
	DBException;

Open and return a database object. If the database does not exist and the openDB flag is false a DBNotFoundException is raised. If the database has been opened before, the previously returned database object is returned again.

Parameters

name

The database name.

create

Flag that determines whether the database is created if it does not exist.

Return Value

The database object.

Exceptions

DBException

Raised if a database failure occurred.

DBNotFoundException

Raised if a database does not exist and the openDB flag is false.

See Also

DB, DB::close

openDBWithTxn

DB openDBWithTxn(bool create,
    string name,
    DBTransaction txn)
    throws
	DBException;

Open and return a database object within the context of a transaction. If the database does not exist and the openDBWithTxn flag is false a DBNotFoundException is raised. If the database has been opened before, the previously returned database object is returned again.

Parameters

txn

The transaction context.

name

The database name.

create

Flag that determines whether the database is created if it does not exist.

Return Value

The database object.

Exceptions

DBException

Raised if a database failure occurred.

DBNotFoundException

Raised if a database does not exist and the openDBWithTxn flag is false.

See Also

DB, DB::close
