Freeze Properties

Freeze.Trace.DB

Synopsis

Freeze.Trace.DB=num

Description

The Freeze database activity trace level:

0No database activity tracing. (default)
1Trace database activity.
2Also trace database transactions.

Freeze.Trace.Evictor

Synopsis

Freeze.Trace.Evictor=num

Description

The Freeze evictor activity trace level:

0No evictor activity tracing. (default)
1Trace Ice object creation and destruction, as well as evictor deactivation.
2Also trace evictor queue activity.
