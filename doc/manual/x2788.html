Ice::NoUserExceptionFactoryException

Overview

local exception NoUserExceptionFactoryException
    extends 
	MarshalException

This exception is a specialization of MarshalException that is raised if no suitable user exception factory was found during user exception unmarshaling.

See Also

UserExceptionFactory, Communicator::addUserExceptionFactory, Communicator::removeUserExceptionFactory, Communicator::findUserExceptionFactory
