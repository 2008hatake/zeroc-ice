Glacier::Starter

Overview

interface Starter

The Glacier router starter.

Operation Index

startRouter

Start a new Glacier router.

startRouter

Router* startRouter(string password,
    out ::Ice::ByteSeq privateKey,
    out ::Ice::ByteSeq publicKey,
    out ::Ice::ByteSeq routerCert,
    string userId)
    throws
	InvalidPasswordException,
	CannotStartRouterException;

Start a new Glacier router. If the password for the given user id is incorrect, an InvalidPasswordException is raised. Otherwise a new router is started, and a proxy to that router is returned to the caller.

Parameters

userId

The user id for which to check the password.

password

The password for the given user id.

privateKey

The RSA Private Key (DER encoded) for the client to use. (Only for SSL.)

publicKey

The RSA Public Key (DER encoded) for the client to use. (Only for SSL.)

routerCert

The trusted certificate of the router. (Only for SSL.)

Return Value

A proxy to the router that has been started.

Exceptions

InvalidPasswordException

Raised if the password for the given user id is not correct.
