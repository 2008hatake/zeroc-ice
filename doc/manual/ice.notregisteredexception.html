Ice::NotRegisteredException

Overview

local exception NotRegisteredException

This exception is raised if an attempt is made to remove a servant, servant locator, facet, object factory, plug-in, object adapter, object, or user exception factory that is not currently registered.

Data Member Index

id

The id (or name) of the object that could not be removed.

kindOfObject

The kind of object that could not be removed: "servant", "servant locator", "facet", "object factory", "plug-in", "object adapter", "object", or "user exception factory".
