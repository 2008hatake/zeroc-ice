Freeze::ObjectStore

Overview

local interface ObjectStore

Provides a means for persistence strategy implementations to save an object's persistent state in the evictor's database.

Used By

PersistenceStrategy::evictedObject, PersistenceStrategy::postOperation, PersistenceStrategy::preOperation

See Also

PersistenceStrategy

Operation Index

save

Saves an object's persistent state.

save

void save(::Ice::Identity ident,
    Object servant)
    throws
	DBException;

Saves an object's persistent state. It is not safe to invoke this operation for a servant with pending mutating operations.

Parameters

ident

The identity of the Ice object.

servant

The servant incarnating the Ice object.

Exceptions

DBException

Raised if a database failure occurred.
