Freeze

Overview

module Freeze

Freeze provides automatic persistence for Ice servants. Freeze provides a binary data format for maximum speed, as well as an XML data format for maximum flexibility. Freeze databases using the XML data format can be migrated when the Slice description of the persistent data changes.

Interface Index

DB

A database of key/value pairs.

DBCursor

A database cursor provides a way to iterate through all key/value pairs in the database.

DBEnvironment

A database environment.

DBTransaction

A transaction object.

EvictionStrategy

Saves the persistent state of an object after eviction of the servant or deactivation of the evictor, but only if the object has been modified by a mutating operation since the last time it was saved.

Evictor

A semi-automatic Ice object persistence manager, based on the evictor pattern.

EvictorIterator

An iterator for the identities managed by the evictor.

IdleStrategy

Saves the persistent state of an object after the invocation of a mutating operation, but only when the object becomes idle (i.e., has no pending mutating operations).

ObjectStore

Provides a means for persistence strategy implementations to save an object's persistent state in the evictor's database.

PersistenceStrategy

A persistence strategy determines when the evictor saves the persistent state of an object.

ServantInitializer

A servant initializer is installed in an evictor and provides the application with an opportunity to perform custom servant initialization.

Exception Index

DBDeadlockException

A Freeze database deadlock exception.

DBException

A Freeze database exception.

DBNotFoundException

A Freeze database exception, indicating that a database record could not be found.

EvictorDeactivatedException

This exception is raised if the evictor has been deactivated.

NoSuchElementException

This exception is raised if there are no further elements in the iteration.

Struct Index

ObjectRecord

The evictor uses a map of ::Ice::Identity to ObjectRecord as its persistent storage.

Statistics

The evictor maintains statistics about each object.

Sequence Index

A database key, represented as a sequence of bytes.

Value

A database value, represented as a sequence of bytes.
