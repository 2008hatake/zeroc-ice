Freeze::IdleStrategy

Overview

local interface IdleStrategy
    extends 
	PersistenceStrategy

Saves the persistent state of an object after the invocation of a mutating operation, but only when the object becomes idle (i.e., has no pending mutating operations). A mutating operation is one that is not defined as nonmutating.

Used By
