readObject

Object readObject(ObjectFactory factory,
    string name,
    string signatureType);

Read a single object. The provided readObject is used if no suitable factory for the object is found and the readObject parameter matches the type of the object being read. If no suitable factory is found, and if the readObject parameter does not match the type of the object that has been read, NoObjectFactoryException is raised.

Parameters

name

The name of the parameter.

signatureType

The signature type for the default object factory.

factory

The default object factory to use if no suitable factory is found and readObject matches the object type.

Return Value

The object that has been read.

See Also

ObjectFactory, readObject
